Documentación

Documentación de componentes de geOrchestra

Desde esta página acceda a la documentación de los diferentes componentes de geOrchestra.

Vista

MapStore2

El visor de datos geográficos avanzado
Plugin Catastro

Un complemento para consultar el registro de la propiedad
Plugin Planificación urbana

Un complemento para consultar documentos urbanísticos

Proporcionar datos

DataFeeder

Un asistente de integración de datos

Publicar

GeoNetwork

Un catálogo de metadatos

GeoServer

Un potente servidor de datos geográficos que respeta los estándares OGC WxS

Data API

Una API de datos abierta que respeta los nuevos estándares “Features API” de OGC

GeoWebCache

Un servidor de caché de mosaicos

Administrar

Console

Una consola de administración para usuarios, roles y organizaciones.

GeoFence

Un módulo de gestión avanzada de derechos de acceso a datos

Analytics

Un módulo que reúne estadísticas de uso de infraestructura

geOrchestra es la infraestructura de datos espaciales gratuita, modular, interoperable y segura, diseñada por la gente para la gente.
